基础格式：
command 【-options】【parameter】
【】代表可选项
Command: 命令
-options： 选项，可以通过选项控制命令行为的细节
parameter:  命令参数， 多用于命令的指向目标

特殊符号：
带“.”的内容为隐藏内容
*通配符：test* : 以test开头的所有内容，*test：以test结尾的所有内容，*test*：包含test内容
｜管道符，如联合grep使用，wc；左边内容作为右边的输入；可以多个连续使用
` xxx ` :   将xxx作为命令输出，如定义的pwd=123， echo    `pwd`;   =>  输出 123
>:   将左侧命令的结果，覆盖写入到符号右侧指定的文件中
>>:  将左侧命令的结果，追加写入到符号右侧指定的文件中
