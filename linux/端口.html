端口是设备与外界通讯交流的入口
端口可以分为物理端口和虚拟端口
物理端口：可见，如USB，HDMI
虚拟端口：不可见，用来操作系统和外部进行交互

linux可以支持65535个端口
公认端口：1～1023，系统内置或知名程序预留，如ssh服务端口22.https端口443，不要占用
注册端口：1024～49151 随意使用，松散的绑定程序，如自己定义的socket
动态端口：程序对外进行网络连接，临时端口，不固定

nmap ip: 查看ip的对外暴露的端口
netstat -anp ｜ grep port:  查看端口占用

yum -y install nmap:  下载nmap
yum -y install net-tools:  下载netstat工具
